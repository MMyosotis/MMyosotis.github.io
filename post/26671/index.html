对前面的数组模拟单向队列的优化，充分利用数组。因此将数组看做是一个环形的，通过取模的方式来实现。

使用 maxSize 代表队列的最大容量，但为了区分队列满和队列空，牺牲了一个存储空间，故队列实际存储长度为 maxSize - 1。

采用变量 front 及 rear 分别记录队列前后端的下标，front 会随着数据输出而改变，而 rear 则是随着数据输入而改变，如图所示：

3.2 队列方法

3.2.1 判断队列空满

判断队列为空的标志是：front = rear。

判断队列为满的标志是：front = (rear + 1) % maxSize。

3.2.2 数据进队列

实现思路为：在 rear 处添加元素，然后 rear = (rear + 1) % maxSize。

3.2.3 数据出队列

实现思路为：输出 front 处元素，清除 front 处元素，最后 front = (front + 1) % maxSize。

3.2.4 获取队列长度

获取队列长度思路为：(rear - front + maxSize) % maxSize，加 maxSize 是为了防止负数。

3.2.4 显示队列

显示队列头的思路为：输出 front 处元素。

显示队列所有数据的思路为：从 front 至 front + getSize() - 1 遍历所有元素并输出，实际下标为 i % maxSize。
