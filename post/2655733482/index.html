Maven - 软件配置

1. 软件下载

Maven 官网地址：https://maven.apache.org/

点击 Download 进入下载页面后，下载对应系统的安装包。

2. 软件安装

下载好的压缩包如下。

将该压缩包打开后里面内容如下，将该文件夹复制到存放软件的目录即可，无需安装。

3. 软件配置

3.1 配置环境变量

右键单击计算机 -> 属性 -> 高级系统设置 -> 环境变量 -> 新建

新建如下系统变量，变量值输入之前解压出的文件夹存放的目录，然后点击确定。

然后双击系统变量 Path，在里面新建如下环境变量，然后点击确定。

3.2 配置本地仓库

打开 maven 文件夹中配置文件： /.../apache-maven-3.8.6/conf/settings.xml

在如下位置插入本地仓库配置项，其中填入本地仓库文件夹的路径，不配置该项则会默认存放于C盘中。

3.3 配置阿里云私服

打开 maven 文件夹中配置文件： /.../apache-maven-3.8.6/conf/settings.xml

查看阿里云私服的配置：阿里云官网-Maven仓库，并复制教程中的内容到如下指定位置。

4. IDEA 配置 Maven

4.1 项目配置

选择 IDEA 里的设置 -> 搜索 Maven -> 点开 Maven 配置界面

进行如下配置，设置 IDEA 使用本地的 Maven，并修改文件配置，此配置将对该项目生效。

4.2 全局配置

在 IDEA 中选择文件 -> 关闭所有项目 -> 进入 IDEA 启动界面 -> 自定义 -> 所有设置

之后的配置过程与项目配置相同，此配置为全局配置，将对所有项目生效。

4.3 创建项目

可以直接创建 Maven 项目

也可以创建空项目 -> 创建新模块 -> 选择 Maven 模块

4.4 导入项目

从侧边栏选中 Maven -> 点击加号 -> 选择要导入的 Maven 项目的 pom.xml 文件

4.5 插件安装

推荐插件：Maven Helper，包含右键单击选择操作和断点调试功能
