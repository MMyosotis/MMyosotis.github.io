板状风致碎片马格努斯效应研究及应用
MQY 2022年10月20日
1.9k 字

板状风致碎片马格努斯效应研究及应用

Keywords

风致碎片，自转，马格努斯效应，气动特性，准定常运动模型

Abstract

  1. 马格努斯效应稳态气动特性风洞实验研究。
  2. 马格努斯效应气动特性随不同转速的变化关系。
  3. 马格努斯效应触发条件数值模拟。
  4. 基于马格努斯效应的准定常运动模型应用。

Introduction

​ 在板状碎片的气动力研究方面，虽然现有的研究手段包括风洞实验或者数值模拟已经能够较好的确定静定气动力，但是对碎片旋转产生的马格努斯效应引起的附加气动力研究还存在较多不足。

​ 在板状碎片飞行特性的研究方面，还大都停留在二维运动的研究中，在三维运动方面的研究较少。国内对这方面的研究开始得更晚，到了近些年才有一些学者进行了尝试(孙晓颖等，2009; 宋芳芳和欧进萍，2010a；宋芳芳和欧进萍，2010b；莫红涛，2012)。Lin et al.（2010a，b）指出，风致碎片对建筑的易损性模型受限于风致碎片运动机理和运动特性研究不足，目前还没有一个较为可靠的评估，因此建立风致碎片飞行特性的预测模型是建立其它相关风灾易损性模型的关键。

​ 基于风致碎片带来的重大损失及目前该领域研究的不足，因此，对风致碎片的运动机理研究以及飞行特性包括运动轨迹、速度等的研究具有显著的理论价值和实际意义。

Method

缩尺风洞试验，CFD 数值模拟，运动方程解析

Main Body

​ 第一章为绪论。首先阐述了本课题的研究背景和研究意义；分别从风洞实验、数值模拟和准定常模型等方面，对风致碎片的运动机理和飞行特性的现有研究进行了回顾与总结；指出了现有文献在风致碎片运动机理和飞行特性方面研究的不足；提出了本文的论文框架和主要研究内容。

​ 第二章为马格努斯稳态气动特性的风洞实验研究。在本章中详细介绍了板状碎片的自转风洞实验设备，并研究了长宽比、宽厚比、雷诺数、转动方向、质量偏心、切角和三维转动对碎片的转速、升力系数、阻力系数和力矩系数的影响；对比本文的实验数据发现了以往文献中预测转速和升力系数的经验公式的不足；利用新数据提出了预测转速和升力系数的新经验公式。

​ 第三章为马格努斯气动特性与转速的关系的风洞实验研究。在本章中利用实验数据进一步研究了长宽比、宽厚比、雷诺数、转动方向、质量偏心、切角和三维转动这些因素对升力系数、阻力系数和力矩系数随转速从 0 到最大转速的变化关系的影响；分析了碎片最大力矩系数与转速的关系；建立了风致碎片从 0 转速到最大转速在各种不同因素影响下的气动力数据库。

​ 第四章为马格努斯效应触发条件的数值模拟。首先结合计算流体动力学（CFD）和刚体动力学（RBD）建立了模拟碎片自转的 CFD-RBD 模型；基于 CFD-RBD 模型研究了长宽比、转动惯量、切角、风速、阻尼力矩和初始释放角度对触发马格努斯效应的最小叶尖速度的影响；基于 CFD-RBD 模拟结果分析了碎片自转过程中涡脱结构与气动力的关系，从原理上解释了碎片发生自转的原因。

​ 第五章为基于马格努斯效应的风致碎片准定常运动模型的应用。结合前几章建立的马格努斯气动力数据库，修正以往模型在低转速区和雷诺数效应方面的不足，建立新的准定常模型；对比本文模型、以往的模型和风致碎片轨迹的实验数据验证本文模型的准确性；利用新模型推导出质量偏心碎片的准定常模型，分析了初始攻角对质量偏心碎片的飞行轨迹的影响；应用新模型修正了以往无量纲水平速度与水平距离的经验公式。

​ 第六章为结论与展望。总结了全文的研究成果，并对未来的进一步研究提出了展望。

Prospect

  1. 在以往的研究中发现，当无量纲转动惯量较大时碎片的马格努斯气动特性与转动惯量无关，但是当无量纲转动惯量较小时碎片的转速、升力系数和阻力系数都对无量纲转动惯量的影响很敏感。本文研究所涉及的工况的转动惯量都较大，而目前碎片无量纲转动惯量较小时的相关研究较少，因此有必要展开对无量纲转动惯量较小时碎片的马格努斯效应的研究。
  2. 在本文建立准定常运动模型的过程中发现了现有三维准定常模型的计算精度较低，本文中虽然对碎片三维马格努斯效应展开了研究，但是由于缺少均匀流下碎片三维飞行的运动轨迹，目前还无法建立更精确的三维准定常模型。碎片三维飞行的运动轨迹实验不仅仅是对认识碎片三维飞行运动原理，还是用来验证相关三维准定常运动模型都起着至关重要的作用。因此，有必要展开进一步的风洞实验，研究风速、起始风攻角、起始倾斜角、长宽比、宽厚比、重量等这些因素对飞行轨迹的影响。
  3. 现有计算风致碎片飞行轨迹的数值模型，包括 CFD-RBD 数值模型和准定常模型都只能模拟均匀流场下碎片的飞行运动，但是在实际的风灾中，碎片大都是在非常复杂的三维湍流场下飞行。因此，有必要对复杂流场下碎片的飞行轨迹展开进一步的研究。
