反射 - Java SE

1. 反射的概念

反射是指对于任何一个 Class 类，在运行的时候都可以直接得到这个类全部成分：

  • 在运行时，可以直接得到这个类的构造器对象：Constructor 类
  • 在运行时，可以直接得到这个类的成员变量对象：Field 类
  • 在运行时，可以直接得到这个类的成员方法对象：Method 类

反射的第一步都是先得到编译后的 Class 类对象，然后就可以得到 Class 的全部成分。

2. 获取类对象

获取 Class 对象共有三种方式：

  • 源码阶段：Class c = forName(String 全限名) ，全限名：包名+类名
  • 对象阶段：Class c2 = 类名.class
  • 运行阶段：创建对象后，Class c3 = 对象.getClass()

3. 获取构造器对象

3.1 获取构造器

先得到类对象，然后从类对象中获取类的构造器对象。Class 类中用于获取构造器 Constructor 的方法：

方法名称 方法说明
Constructor[] getConstructors() 返回所有构造器对象的数组（只能拿public的）
Constructor[] getDeclaredConstructors() 返回所有构造器对象的数组，存在就能拿到
Constructor getConstructor(Class... parameterTypes) 返回单个构造器对象（只能拿public的）
Constructor getDeclaredConstructor(Class...parameterTypes) 返回单个构造器对象，存在就能拿到

3.2 使用构造器

获取构造器对象后，可以创建一个对象返回。Constructor 中用于创建对象的方法如下：

方法名称 方法说明
T newInstance(Object... initargs) 根据指定的构造器创建对象
public void setAccessible(boolean flag) 设置为true，表示取消访问检查，进行暴力反射，可以使用私有的成员

使用构造器创建对象的注意事项如下：

  • 对于私有构造器，必须先使用暴力反射打开权限，才能使用构造器创建对象。
  • 使用 newInstance 创建对象的类型为 Object，可以将其强转成实际类型。

4. 获取成员变量对象

4.1 获取成员变量

先得到类对象，然后从类对象中获取类的成员变量对象。Class 类中用于获取成员变量 Filed 的方法如下：

方法名称 方法说明
Field[] getFields() 返回所有成员变量对象的数组（只能拿public的）
Field[] getDeclaredFields() 返回所有成员变量对象的数组，存在就能拿到
Field getField(String name) 返回单个成员变量对象（只能拿public的）
Field getDeclaredField(String name) 返回单个成员变量对象，存在就能拿到

4.2 使用成员变量

获取成员变量对象后，可以创建对象并在该对象中取值、赋值。Field 中用于取值和赋值的方法如下：

方法名称 方法说明
void set(Object obj, Object value) 赋值
Object get(Object obj) 获取值
public void setAccessible(boolean flag) 设置为true，表示取消访问检查，进行暴力反射，可以使用私有的成员

对成员变量赋值和取值的注意事项如下：

  • 需要先创建对象，然后再在该对象中操作取值和赋值。
  • 对于私有成员变量，必须先使用暴力反射打开权限，才能赋值和取值。
  • 使用 get 获取值时返回类型为 Object 类型，可以将其强转成实际类型。

5. 获取方法对象

5.1 获取方法

先得到类对象，然后从类对象中获取类的方法对象。Class 类中用于获取方法 Method 的方法如下：

方法名称 方法说明
Method[] getMethods() 返回所有成员方法对象的数组（只能拿public的）
Method[] getDeclaredMethods() 返回所有成员方法对象的数组，存在就能拿到
Method getMethod(String name, Class... parameterTypes) 返回单个成员方法对象（只能拿public的）
Method getDeclaredMethod(String name, Class... parameterTypes) 返回单个成员方法对象，存在就能拿到

5.2 使用方法

获取方法对象后，可以创建对象并在该对象中执行方法。Method 中用于除法执行的方法如下：

方法名称 方法说明
Object invoke(Object obj, Object... args) 运行方法
参数一：调用该方法的对象
参数二：调用方法的传递的参数（如果没有就不写）
返回值：方法的返回值（如果没有就不写）
public void setAccessible(boolean flag) 设置为true，表示取消访问检查，进行暴力反射，可以使用私有的成员

使用方法的注意事项如下：

  • 需要先创建对象，然后再在该对象中执行方法。
  • 对于私有方法，必须先使用暴力反射打开权限，才能触发执行。
  • 使用 invoke 执行方法，对于有返回值的方法，返回类型为 Object 类型，可以将其强转成实际类型。

6. 反射的应用举例

6.1 绕过集合泛型添加数据

泛型只是在编译阶段可以约束集合只能操作某种数据类型，在编译成 Class 文件进入运行阶段的时候，其真实类型都是 ArrayList 了，泛型相当于被擦除了。

反射是作用在运行时的技术，此时集合的泛型将不能产生约束了，此时是可以为集合存入其他任意类型的元素的。

6.2 通用框架的底层原理

通过反射可以实现：对任意一个对象，在不清楚对象字段的情况可以，可以把对象的字段名称和对应值存储到文件中去。

实现步骤如下：

  • 定义一个方法，可以接收任意类的对象。

  • 每次收到一个对象后，使用反射获取对象的Class类对象，然后获取全部成员变量信息。

  • 遍历成员变量信息，然后提取本成员变量在对象中的具体值。

  • 存入成员变量名称和值到文件中去。
