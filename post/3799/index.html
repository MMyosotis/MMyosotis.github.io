1.5 Arrays 工具类

1.5.1 基础方法

Arrays 是在 java.util 包下的一个工具类，其中常用方法如下：

方法名称 方法说明
public static String toString(Object[] a) 返回指定数组内容的字符串表示形式。
public static boolean equals(Object[] a, Object[] a2) 判断两个数组中的元素是否一一对应相等。
public static int binarySearch(int[] a, int key) 二分搜索数组中的数据（必须排好序），存在则返回索引，不存在返回-1
public static <T> T[] copyOf(T[] original, int newLength) 将原始数组的元素，复制到新的数组中，可以设置复制的长度

数组直接打印是打印地址值，只有调用 toString 方法才是打印数组的内容。

对于 CopyOf 方法有如下说明：

  • 若新数组长度大于拷贝长度，则额外的元素将被赋予默认值。
  • 如果新数组长度小于拷贝长度，则只拷贝前面的值。

1.5.2 排序方法

数组排序的方法

Arrays 提供了针对数组排序的方法：

方法名称 方法说明
public static void sort(Object[] a) 对数组元素按照升序排序
public static <T> void sort(类型[] a, Comparator<? super T> c) 使用比较器自定义排序

自定义比较器即设置 Comparator 接口对应的比较器对象，来定制比较规则，此时排序数组的类型必须是引用类型。对于自定义类型，则无法使用默认 sort 排序，必须自定义比较器。

比较器返回值规则

比较器 Comparator 返回值的规则为：（该返回值如果是 Double 类型强转 int 的话要特别注意）

  • 如果认为第一个元素大于第二个元素：返回正整数即可。
  • 如果认为第一个元素小于第二个元素：返回负整数即可。
  • 如果认为第一个元素等于第二个元素：返回 0 即可。

可以参见 TreeSet 排序，现给出 sort 方法中自定义比较器的使用例子。

2. 正则表达式

2.1 基本语法

语法

  • 判断是否匹配正则表达式

    public boolean matches(String regex)：匹配返回 true ，不匹配返回 false ，此处匹配指整个字符串要匹配上。

特殊符号

正则表达式中特殊符号如下：

  • 通配符：可以匹配多个不同字符的特殊字符

    ​ 带 \ 的符号在使用时，需要使用 \\ 。前一个 \ 表示字符串中转义，后一个表示 \ 符号本身，即对正则表达式转义。

    符号 说明
    . 匹配除换行符\n之外的所有字符
    \d 匹配从0到9的一个数字，即 [0-9]
    \D 匹配非数字，即 [^0-9]
    \s 匹配单个空格符，包括 tab 键和换行符，即 [\t\n\x0B\f\r]
    \S 匹配除单个空格符之外的所有字符，即 [^\s]
    \w 匹配字母，数字或下划线字符，即 [a-zA-Z_0-9]
    \W 匹配所有与\w不匹配的字符，即 [^\w]

  • 边界符：用于匹配开头或结尾

    符号 说明
    ^ 匹配一行字符串开头位置
    $ 匹配一行字符串结尾位置

  • 范围符：[] 匹配字符集合，允许使用者在匹配模式中指定多个字符或某个范围，[^] 规定目标对象中不能存在模式中所规定的字符串。

    符号 说明
    [abc] a，或 b，或 c
    [^abc] 除了 a，b，c 以外的任何字符
    [^abc] 除了 a，b，c 以外的任何字符
    [a-zA-Z] a 到 z，或 A 到 Z
    [a-d[m-p]] a 到 d，或 m 到 p
    [a-z&&[def]] d，或 e，或 f
    [a-z&&[^bc]] a 到 z，除了 b 和 c
    [a-z&&[^m-p]] a 到 z，除了 m 到 p

  • 频率符：可以指定模式在匹配对象中出现的频率。

    符号 说明
    + 其前导字符必须在目标对象中连续出现一次或多次
    * 其前导字符必须在目标对象中出现零次或连续多次
    ? 其前导字符必须在目标对象中连续出现零次或一次
    {n} 其前导字符正好出现 n 次
    {n,} 其前导字符至少出现 n 次
    {n,m} 其前导字符至少出现 n 次，但不超过 m 次

  • 同时符：() 符号包含的内容必须同时且连续出现在目标对象中。

  • 逻辑符：| 用于连接多个条件，表示逻辑或

注意

  • Java 中 \ 为转义符，正则表达式中 \ 也为转义符。

  • 字符串中要输出特殊符号本身必须用 \符号，正则表达式中要输出特殊符号本身必须用 \\符号。

  • 字符串中输出 \ 符号本身必须使用 \\，正则表示中输出 \ 符号本身必须使用 \\\\

2.2 表达式应用

在字符串中的应用

正则表达式可以在调用 String 的方法时使用：

方法名称 方法说明
public String replaceAll(String regex,String newStr) 按照正则表达式匹配的内容进行替换
public String[] split(String regex) 按正则表达式匹配的内容进行分割字符串，反回一个字符串数组

正则表达式的编译形式

指定为字符串的正则表达式必须首先被编译为此类的实例。然后可将得到的 Patter 匹配对象用于创建 Matcher 对象，依照正则表达式，该对象可以与任意字符序列匹配。执行匹配所涉及的所有状态都驻留在匹配器中，所以多个匹配器可以共享同一模式。

之后可以通过调用匹配器的方法进行后续操作。

3. 内存分配

Java 内存分配共有五个分区：

  • 栈：方法运行时使用的内存，比如 main 方法运行，进入方法栈中执行。
  • 堆：存储对象或者数组，new 来创建的都存储在堆内存。堆中开辟的空间均有地址值。
  • 方法区：存储可以运行的 class 文件。
  • 本地方法栈：JVM 在使用操作系统功能的时候使用，和我们开发无关。
  • 寄存机：给 CPU 使用，和我们开发无关。
