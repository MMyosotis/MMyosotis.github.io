基础语法 (一) - Java SE

1. 数据类型

1.1 二进制存储

计算机的数据单位

计算机在底层都是采用二进制，使用0和1来存储，其数据单元如下：

  • 位：字节中的每个二进制位就称为位 bit，简称 b。二进制使用最高位表示符号位，用1表示负数，用0表示正数。
  • 字节：计算机中数据最小的组成单元成为字节 byte，简称 B。其由8个二进制位组成。

进制的转换

十进制转二进制分为整数和小数两种形式：

  • 整数二进制转换：除2取余，逆序排列，直到商为0为止。

  • 小数二进制转换：乘2取整，正序排列，直到小数为0或达到指定精度为止。

由二进制数转换成十进制数的基本做法是，把二进制数首先写成加权系数展开式，然后按十进制加法规则求和。

示例：

以二进制数110.101为例，(110.101)2=1×22+1×21+0×20+1×2-1+0×2-2+1×2-3=(6.625)10

1.2 基本数据类型

基本数据类型

基本数据类型 存储需求 默认值 取值范围
整型 byte 1字节 （8位） 0 (-27, 27-1)
整型 short 2字节 （16位） 0 (-215, 215-1)
整型 int 4字节 （32位） 0 (-231, 231-1)
整型 long 8字节 （64位） 0 (-263, 263-1)
浮点型 float 4字节 （32位） 0.0f 近似认为 (-2128, 2^128) （IEEE 754）
浮点型 double 8字节 （64位） 0.0d 近似认为 (-21024, 2^1024) （IEEE 754）
字符型 char 2字节 （16位） \u0000 (即0) (0, 2^16)
布尔型 boolean 1字节 （8位） false 0, 1

整型数据存储

整型数据存储由符号位，数据位构成。以 byte 类型为例，其存储的最大正数为127，而最小负数为-128，这是因为+0用来存储0，而-0用来存储了-128，其他整型类型也是如此。

浮点型数据存储

浮点型数据存储由符号位(S)，指数位(E)，有效数字位(M)构成，即(-1)S × M × 2E。

float 类型符号位1位，指数位8位，有效数字位23位。double 类型符号位1位，指数位11位，有效数字位52位。

java 中浮点数符合 IEEE 754 标准：

  • 符号位(S)：正数为0，负数为1。
  • 指数位(E)：指数实际取值范围为 (-126, 127) 或 (-1022, 1023)。
    • 规范化数：指数E不为全0或全1，默认小数点前为1。E的十进制值减127（或1023），即得真实指数值。
    • 非规范化数：指数E全为0，默认小数点前为0。E等于1-127（或1-1023）。
    • 特殊数值：指数E全为1。表示特殊值。
  • 有效数字位(M)：M只存储小数点后的数字。
描述 指数位 小数位 单精度 双精度
0 00...00 0...00 0 0
1 01...11 0...00 1×20 1×20
∞ 11...11 0...00 ∞ ∞
最小非格式化数 00...00 0...01 2-23×2-126≈1.4E-45 2-52×2-1022
最大非格式化数 00...00 1...11 (1-2-23)×2-126≈1.18E-38 (1-2-52)×2-1022
最小格式化数 00...01 0...00 1×2-126≈1.18E-38 1×2-1022
最大格式化数 11...10 1...11 (2-2-23)×2127≈3.4E+38 (2-2-52)×21023

示例：

以浮点数6.625f的存储为例，其二进制表示为110.101，将其规范化表示为1.10101×2^2。

有效数字位：因为规范化表示后小数点前必须为1，所以有效数字位只存储小数点后的数字，即10101000000000000000000。

指数位：指数为2，加上127之后为129，即10000001。

符号位：正数为0。

最终存储结果：0 10000001 10101000000000000000000

浮点型在转换为二进制时，存在无法精确转换的问题（乘2取整但一直无法使小数为0），那么在执行浮点数操作时（算数运算、比较运算等），要考虑精度丢失可能带来的问题。可以通过 BigDecimal 类解决。

1.3 引用数据类型

引用数据类型包括数组、类 、接口三大类，其默认值均为 null 。

每个基本数据类型均有其对应的引用类型，也被称为包装类。

1.4 类型转换

自动类型转换

变量和表达式均可进行自动类型转换：

  • 变量自动类型转换：

    类型范围小的变量，可以直接赋值给类型范围大的变量，反之则不能自动转换。

    其类型自动转换的底层原理如下图：

    其他形式的还有 byte → short → int → long → float → double 和 char → int。

  • 表达式自动类型转换：

    在表达式中，小范围类型的变量会自动转换成当前较大范围的类型再运算。

    表达式的最终结果类型由表达式中的最高类型决定。

    在表达式中，byte、short、char 是直接转换成 int 类型参与运算的。

强制类型转换

可以强制将类型范围大的变量、数据赋值给类型范围小的变量。只需要在转换的变量前加上括号，括号内为想要转换的类型。

2. 运算符

2.1 算数运算符

运算符 说明
+ 加
- 减
* 乘
/ 除
% 取余

运算符的注意事项如下：

  • 如果是整型数据进行除法运算，其结果必然是整型数据，小数部分会自动抹掉（是抹掉不是四舍五入）。

  • 多个字符串允许用 + 号拼接，在遇到字符前的 + 负责相加数字，对于其后紧跟字符、以及这之后的 + 均表示拼接字符。

  • 整数数字直接用运算符计算时，默认是 int 类型，如果运算过程中出现结果过大超过 int 存储范围，则会导致结果失真，要注意手动将数字定为 long 类型。

2.2 自增自减运算符

运算符 说明
++ 自+1
-- 自-1

自增自减运算符放于变量前后的区别如下：

  • 放在变量的前面：先对变量进行+1、-1，再拿变量的值进行运算。
  • 放在变量的后面：先拿变量的值进行运算，再对变量的值进行+1、-1 。

2.3 赋值运算符

赋值运算符

赋值运算符包括：=

赋值运算符把右边的值赋给左边，且赋值运算符还自带了强制类型转换。

二元运算符

二元运算符包括：+=、-=、*=、/=、%=

二元运算符将计算和赋值合并书写，a += b 等同于 a = (a+b)。

2.4 关系运算符

关系运算符是对数据进行条件判断的符号，最终会返回一个比较的 boolean 结果。关系运算符如下：

运算符 说明
== a == b，判断 a 是否等于 b
!= a != b，判断 a 和 b 是否不相等
> a > b，判断 a 是否大于 b
< a < b，判断 a 是否小于 b
>= a >= b，判断 a 是否大于等于 b
<= a <= b，判断 a 是否小于等于 b

2.5 逻辑运算符

逻辑运算符

运算符 说明
& 逻辑与（无论左边结果，右边都会执行）
| 逻辑或（无论左边结果，右边都会执行）
! 逻辑非
^ 逻辑异或，两个条件不同结果则为true，否则为false
&& 短路与（左边为 false 则不执行右边）
|| 短路或（左边为 true 则不执行右边）

按位运算符

如果进行比较的双方是数字的话，那么用如下比较符进行比较就会变成按位运算。

按位运算即针对里面每一位均进行比较，每位的结果均为0或1，最终结果再由二进制转换为十进制。

运算符 说明
& 按位与
| 按位或
~ 按位非
^ 按位异或

2.6 移位运算符

运算符 说明
<< 左移（丢弃左边指定位数，右边补0）
>> 右移（丢弃右边指定位数，左边补最高位的值）
>>> 无符号右移（丢弃右边指定位数，左边补0）

2.7 三元运算符

三元运算符语法为：条件表达式? 表达式1: 表达式2

首先计算关系表达式的返回值，如果值为 true 则调用表达式1，如果为 false 则调用表达式2。

3. 输入与输出

3.1 控制台输入

输入使用 Scanner 类。

标准输入流 Scanner 类常用的方法如下：

方法名称 方法说明
String nextLine() 读取输入的下一行内容
String next() 读取输入的下一个单词（以空格作为分隔符）
int nextInt() 读取并转换下一个表示整数的字符序列
int nextDouble() 读取并转换下一个表示整数或浮点数的字符序列

System.in 实际返回的是一个 InputStream 指向控制台的字节输入流。

3.2 控制台输出

输出语法

System.out.println() ：输出括号中的内容并换行。

System.out.print() ：输出括号中的内容且不换行。

System.out.printf() ：输出括号中的内容且不换行，允许使用格式字符如下：

格式字符 功能
%n 换行，相当于 \n
%c 单个字符
%d 十进制整数
%u 无符号十进制数
%f 十进制浮点数，可以指定位数如 %3.2f ，两个数字分别为小数点前后显示的位数
%o 八进制数
%x 十六进制数
%s 字符串
%% 输出百分号

转义序列

Java 中输出特殊符号需要用到转义序列如下：

转义序列 名称
\b 退格
\t 制表
\n 换行
\r 回车
\" 双引号
\' 单引号
\\ 反斜杠

System.out 实际返回的是一个 PrintStream 默认指向控制台的打印流。

4. Java 控制流程

4.1 条件语句

4.1.1 if else 语句

if 语句

if 语句可以单独使用，当 if 后条件语句为真则执行大括号里的内容。若大括号内只有一条语句，则大括号可以不写。

if..else 语句

if 语句也可以和 else 语句连用，if 语句执行了就不会再执行 else 语句了。

if…elseif 语句

存在多个选择分支时，可以用用 if...elseif 语句，按顺序从上往下匹配，其中任意一句执行后都不会再执行其他的分支。

4.1.2 switch 语句

switch 语句根据执行条件去匹配不同分支，switch 语句书写的注意事项如下：

  • 若不在分支后写 break，则执行完该分支后会直接执行下一个 case 中的代码块（不论是否匹配上），直到遇到 break。
  • switch 后跟的表达式类型只能是 byte、short、int、char、枚举 ( JDK5后)、String ( JDK7后)，不支持 double、float、long。
  • case 给出的值不允许重复，且只能是字面量，不能是变量。

4.2 循环语句

4.2.1 while 循环

while 循环

while 语句会在括号中表达式为真时一直反复执行大括号内的语句。

可以通过 while (true) 这种写法来一直执行循环。

do while 循环

do while 循环与 while 循环的区别在于，do while 是先执行再判断，大括号内的语句至少会执行一次。

4.2.2 for 循环

for 循环

for 循环的语法为：for (初始化; 布尔表达式; 步进) {} ，每次判断表达式为真后就会指定代码块，然后执行一次步进的语句。

for each 循环

for each 循环的语法为：for (变量 : 集合) {} ，变量用于挨个接受元素，可以用来依次遍历数组或者集合这种结构。

4.3 跳转语句

4.3.1 break 语句

break 用于跳出并结束当前所在循环。

4.3.2 continue 语句

continue 用于当次循环的执行，并开始下一次循环。

4.3.3 return 语句

return 语句可以从一个方法返回，并把控制权交给调用它的语句。
