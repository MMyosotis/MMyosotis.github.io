基于 Fluid 主题添加动画

1. 前言

从搭建好博客没过多久我就开始嫌弃他几乎没有什么过度动画了，虽然对于 Fluid 主题而言这样平铺直叙的页面出现反而更显简洁优雅，但我确实喜欢简中带点花里胡哨，只能撸起袖子自己来了。

在添加动画这部分，Fluid 主题下确实没找到我能参考的文章，最后只能去一遍又一遍看别人的网页源码，搜索 transition、animation 之类的属性，看一看大概是怎么写的，然后找了网上不少教程和网课又稍微看了看，最后才整出了一套能满足我基本需求的动画。

2. 通用动画

通用动画

因为有一部分动画会在之后不同页面的设置中反复用到，所以就写在最前面，这里用到的是 @keyframes。

这里特地把透明度变化的动画单独写出来，是因为后面有不少地方想要做一个渐入效果，但是透明度变化与方位变化的时间并不统一，所以干脆就直接分成两个动画写，各自设置各自的时间。

通用页面进入

为了使所有的页面进入时都更加平滑，所以添加了一个统一的页面进入动画，这里就用到了前面写的透明度变化的动画，然后再额外加了一个向上移动，同时设置了 animation-fill-mode:both，使页面渐入前不要显示出来，渐入后停在原位。同时有一个上方导航栏向下的渐入动画。

3. 主页动画

因为想要在鼠标滑动的时候更有选择的感觉一些，所以设置了一个鼠标悬浮放大的动画，这里用到了伪类 :hover ，用于选择鼠标指针浮动在上面的元素。

手机端为了流畅就没有设置页面进入等相关动画，取而代之的是增加了文字进入动画，通过 css 媒体查询 @media 来实现多端设置。

4. 归档页动画

归档页除了保留本身 Fluid 主题自带的鼠标悬浮换色的动画以外，还额外添加了鼠标悬浮放大，同样用到了伪类 :hover 。然后调大了一点字体，针对不同大小的设备又做了一些调整。

手机端为了流畅就没有设置页面进入等相关动画，取而代之的是增加了文字进入动画，通过 css 媒体查询 @media 来实现多端设置。

5. 分类页动画

分类页和归档页一样，除了保留本身 Fluid 主题自带的鼠标悬浮换色的动画以外，还额外添加了鼠标悬浮放大，同样用到了伪类 :hover 。然后调大了一点字体，针对不同大小的设备又做了一些调整。

手机端为了流畅就没有设置页面进入等相关动画，取而代之的是增加了文字进入动画，通过 css 媒体查询 @media 来实现多端设置。

6. 关于页动画

除了基本的页面进入动画以外，关于页因为有头像，所以额外设置了一个头像的入场动画，这样显得比较有层次感。

7. 文章页动画

文章页的目录设置了一个从左向右的入场动画，设置了 animation-delay: 0.5s ，会比文章页面整体进入要晚一点。

手机端为了流畅就没有设置页面进入等相关动画，取而代之的是增加了文字进入动画，通过 css 媒体查询 @media 来实现多端设置。

文章页的标题没有使用打字机效果，所以增加了一个从上到下的入场动画，和页面整体入场是同时进行的。

8. 自定页动画

自定义页并未再单独做动画，而是直接和归档页的动画一起设置，这主要还是为了风格统一。
